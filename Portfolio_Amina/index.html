Application web de réservation d'un restaurant

Année : 2021

Développement Android

Informations

Développement via Android Studio d’une application Web de réservation dans un restaurant pour les personnes à faible revenu. Le client choisit la ville et le type de restaurant (Chinois, Indien, Japonais, etc..), l’application affiche tous les résultats liés à sa recherche. Une fois le restaurant choisit par le client tous les détails sont affichés (Menu, Horaire, Numéro de téléphone, etc..) et le client en ce moment doit choisir son repas et entrer ses informations personnelles pour la réservation. Un message de confirmation est envoyé à son courriel
